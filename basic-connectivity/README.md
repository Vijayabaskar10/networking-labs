# Basic 2-PC Connectivity Lab

## Objective
Connect two PCs and verify communication using ping.

## Devices Used
- 2 PCs
- Copper Cross-over Cable

## IP Addressing

| Device | IP Address |
|--------|-------------|
| PC1 | 10.10.10.1 |
| PC2 | 10.10.10.2 |

## Verification
Ping successful between both PCs.

## Concepts Learned
- IP addressing
- Subnet mask
- ICMP ping
- Basic connectivity troubleshooting
